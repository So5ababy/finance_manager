Finance Manager 
1. Запуск
- Соберите проект: mvn package
- Запустите: java -jar target/finance-manager-1.0-SNAPSHOT.jar

2. Основные команды
   - Главное меню:
   - 1 – регистрация
   - 2 – вход
   - 3 - выход
- Меню кошелька (после входа или регистрации):
   - 1 – показать баланс
   - 2 – добавить доход
   - 3 – добавить расход
   - 4 - просмотр транзакций
   - 5 - добавить категорию с бюджетом
   - 6 - показать информацию по категориям
   - 7 - показать сводку по кошельку
   - 8 - изменить бюджет по категории
   - 9 - удалить категорию
   - 10 - выйти из кошелька
3. Структура проекта
- Main.java - точка входа
- Wallet.java - логика кошелька
- User.java - модель пользователя
- UserSettings.java - работа с пользователями
- Memory.java - сохранение/загрузка JSON
- WalletInterface.java - CLI

4. Как работает
- При запуске загружаются пользователи из users.json
- После входа загружается кошелёк пользователя из login_wallet.json 
- Все изменения сохраняются при выходе
